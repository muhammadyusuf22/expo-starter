Spenduit Lite - Aplikasi Pencatat Keuangan Pribadi

1. Pendahuluan

Kebijakan Privasi ini menjelaskan bagaimana Spenduit Lite ("Aplikasi", "kami") menangani informasi pengguna. Kami berkomitmen untuk melindungi privasi Anda dan memastikan transparansi dalam pengelolaan data.

2. Data yang Kami Kumpulkan

Spenduit Lite tidak mengumpulkan data pribadi apapun.

Aplikasi ini dirancang untuk bekerja sepenuhnya secara offline. Semua informasi yang Anda masukkan, termasuk:

  • Catatan transaksi (pemasukan dan pengeluaran)
  • Informasi wallet/dompet
  • Target tabungan (goals)
  • Kategori kustom

Tersimpan hanya di perangkat Anda dan tidak pernah dikirim ke server manapun.

3. Penyimpanan Data

Semua data disimpan secara lokal menggunakan database SQLite di perangkat Anda. Kami tidak memiliki akses ke data ini dan tidak dapat melihat, mengubah, atau menghapusnya.

4. Berbagi Data dengan Pihak Ketiga

Kami tidak membagikan data Anda dengan pihak ketiga manapun karena kami tidak mengumpulkan data dari pengguna.

5. Keamanan Data

Karena semua data tersimpan lokal di perangkat Anda:

  • Keamanan data bergantung pada keamanan perangkat Anda
  • Kami menyarankan untuk menggunakan kunci layar dan enkripsi perangkat
  • Jika Anda menghapus aplikasi, semua data akan ikut terhapus

6. Hak Pengguna

Anda memiliki kontrol penuh atas data Anda:

  • Akses: Semua data dapat dilihat langsung di aplikasi
  • Ubah: Anda dapat mengubah atau menghapus transaksi kapan saja
  • Hapus: Menghapus aplikasi akan menghapus semua data

7. Izin Aplikasi

Spenduit Lite tidak memerlukan izin khusus yang berkaitan dengan privasi seperti akses ke kontak, kamera, lokasi, atau penyimpanan eksternal.

8. Anak-anak

Aplikasi ini aman digunakan oleh semua usia. Karena tidak ada pengumpulan data, tidak ada risiko privasi khusus untuk pengguna di bawah umur.

9. Perubahan Kebijakan

Jika ada perubahan pada kebijakan privasi ini, kami akan memperbarui halaman ini dan mencantumkan tanggal pembaruan di bagian bawah.
